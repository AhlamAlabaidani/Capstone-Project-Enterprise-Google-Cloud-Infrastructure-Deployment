# Capstone-Project-Enterprise-Google-Cloud-Infrastructure-Deployment
# GCP Capstone Project

## Overview
This project deploys a secure and scalable cloud infrastructure on Google Cloud Platform.

## Architecture
- VPC with public and private subnets
- Two Compute Engine VMs in private subnet
- HTTP Load Balancer for traffic distribution
- Firewall rules for controlled access
- Infrastructure as Code using Terraform

## Tools Used
- Google Cloud Platform
- Compute Engine
- VPC Network
- Cloud Load Balancing
- Terraform
- GitHub

## Features
- Secure private VM deployment
- Load balancing across multiple instances
- Automated infrastructure provisioning
- Scalable architecture design

## Status
Project successfully deployed and tested.
